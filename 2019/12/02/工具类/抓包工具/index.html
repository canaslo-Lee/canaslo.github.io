charles

第一步：

官网下载charles；官网地址： https://www.charlesproxy.com/

右边 找DOWNLOAD 点击下边的Download a free trial Version

根据你的pc系统选择对应的安装包下载

下载完成后下一步安装即可；可自行配置软件安装位置

该软件是歌付费软件；由于本人支持正版；所以坚定的使用试用版

试用版和付费版的区别是试用版有一个等待时间

第二步

根据你的安装目录找到这个文件夹；进入后根据你的pc系统选择软件启动文件双击启动；为方便操作，建议添加到桌面或者开始屏幕；双击打开软件

界面功能简单介绍：

工具栏中 proxy（代理）；下边一栏左一的刷子是清空

工具栏依次查找：help –> SSL proxying –> install charles root certificate on a mobile device or remote browser 选中 会出现弹窗

02.一元操作符

阅读量: 静思 2019-12-03 10:18:12
Categories： javascript Tags： js基础

描述

  • 只操作一个值的操作符
  • 是es中最简单的操作符

1. 递增/减操作符

  • 描述：

    • 递增（++）/减（–）照搬自C语言
    • 有前缀版和后缀版两个版本

  • 前缀版：位于要操作的变量前

    let age = 29
++age // 29 + 1 = 30
// 等于下列代码
age = age + 1 // 30

--age // 30 -1 = 29
// 等于下列代码
age = age - 1 // 29
    • 前缀递增/减变量的值都会在语句求值前改变（计算机科学中被称为有副作用）
      let age = 29
let anotherAge = --age + 2
console.log(age)
  // 29 - 1 = 28
console.log(anotherAge)
  // 29 - 1 + 2 = 30
    • 前缀递增/减在语句中优先级相同，因此计算方式是由左到右
      let n1 = 2
let n2 = 20
let n3 = --n1 + n2
let n4 = n1 + n2
console.log(n3)
  // 2 - 1 + 20 = 21
console.log(n4)
  // 2 - 1 + 20 = 21 // 因为此处使用的是递减后的值

  • 后缀版：位于要操作的变量后

    let age = 29
age++ // 29 + 1 = 30
    • 与其他操作混合时会出现明显差异
      let n1 = 2
let n2 = 20
let n3 = n1-- + n2
let n4 = n1 + n2
console.log(n3)
  // 2 + 20 = 22 // 此处的求值使用的是n1的原始值（2）
console.log(n4)
  // 2 - 1 + 20 = 21 // 此处求值使用的是n1递减后的值（1）

  • 区别： - 后缀版：递增/减在语句求值后改变（使用递增/减前的值计算，下次计算使用递增/减后的值计算） - 前缀版：递增/减在语句求值前改变（使用递增/减前的值计算）

  • 上述四个操作符可用作任何值，不限于整数、字符串、布尔值、浮点值、对象

  • 递增/减遵循以下规则：

    • 字符串
      • 有效数值：转为数值再应用改变，变量类型从字符串变为数值
      • 非有效值：变量值设置位NaN，变量类型从字符串变为数值
    • 布尔值
      • false：转为0再应用改变，变量类型从布尔值变为数值
      • true：转为1再应用改变，变量类型从布尔值变为数值
    • 浮点值
      • 加/减1
    • 对象
      • 调用valueOf()方法获取可操作值
      • 对得到的值执行上述规则
      • 若得到的是NaN，则调用toString()并再次应用其规则
      • 变量类型从对象变为数值
        let s1 = "2"
let s2 = "z"
let b = false
let f = 1.1
let o = { valueOf() { return -1 } }
s1++
s2++
b++
f--
o--
console.log(s1)
// 3
console.log(s2)
// NaN
console.log(b)
// 1
console.log(f)
// 0.10000000000000009 // 浮点数存在精度丢失问题
console.log(o)
// -2

2. 一元加/减

  • es中和高中数学用途一致

  • 由一个加号（+）/减号（-）表示

  • 对数值无任何影响

    // 一元加
let num = 25
num = +num
console.log(num)
  // 25

// 一元减
num = -num
console.log(num)
  // -25

  • 应用在非数值上，会执行使用Number()转型函数一样的类型转换

    • 布尔值false/true转为0/1
    • 字符串根据特殊规则解析
    • 对象调用其valueOf()、/、toString()方法得到可转换的值
      let s1 = "01"
let s2 = "1.1"
let s3 = "z"
let b = false
let f = 1.1
let o = { valueOf() { return -1 } }
s1 = +s1
s2 = +s2
s3 = +s3
b = +b
f = +f
o = +o
console.log(s1)
  // 1
console.log(s2)
  // 1.1
console.log(s3)
  // NaN
console.log(b)
  // 0
console.log(f)
  // 1.1
console.log(o)
  // -1
s1 = -s1
s2 = -s2
s3 = -s3
b = -b
f = -f
o = -o
console.log(s1)
  // -1
console.log(s2)
  // -1.1
console.log(s3)
  // NaN
console.log(b)
  // -0
console.log(f)
  // -1.1
console.log(o)
  // 1

  • 一元加/减主用于基本算术或数据类型转换

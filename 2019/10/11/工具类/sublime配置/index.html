sublime配置

阅读量: 李宜珂 2019-10-11 10:23:00
Categories： Tags： 工具类

sublime常用插件

autobackups 代码备份功能：可以查找旧版本的代码，可在其设置中更改备份位置

docblockr 注释插件：方法前输入 /** 可以快捷生成注释模板

emmet html 快捷输入标签

sublime server localhost 的方式打开html文件

DeleteBlankLines 删除空行 可使用ctrl + a 全选 ctrl + alt + backspace 将全部的空行删除

jsformat js代码格式化插件

保存自动格式化

先下载jsformat插件

下载完成之后选择该插件的配置文件：首选项-》package settings-》jsformat-》settings-user 双击打开 输入：

{
    "e4x":true,
    //jsformat options
    "format_on_save":true,
}

输入完成后点击保存，即可正常ctrl+s自动格式化保存js文件

永久自动换行

打开sublime 选择首选项 设置 ctrl+f查找word_wrap是否存在

右侧一般都会有个user的文件，在最外层的花括号{} 内最后一个配置的结尾无逗号时，添加一个逗号，换行写入：”word_wrap”: true

保存即可

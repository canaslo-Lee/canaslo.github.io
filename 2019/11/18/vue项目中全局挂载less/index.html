vue项目中全局挂载less

李宜珂 2019-11-18 19:46:16
Categories： Tags： vue

vue中全局挂载less

项目搭建完成后使用cmd 运行命令

vue add style-resources-loader

添加完成后 在 vue.config.js内进行配置

// url: 全局挂载的less文件在项目内的位置 
const path = require('path')
module.exports = {
    pluginOptions: {
        'style-resources-loader': {
            preProcessor: 'less',
            patterns: [
                path.resolve(__dirname, 'url')
            ]
        }
    }
}

入口文件内引入 全局挂载的less文件

import 'urlPath'

重启项目 重启项目 重启项目！！！

在公共less内随便写点啥

@red: red;
@abstl0: {
    position:abslute;
    top:0;
    left:0;
}

随便找个.vue文件内使用

.container{
    color: @red;
    @abstl0;
}

解释一下 @abstl0; 的意思：将全局less文件内定义的样式插入到这里

保存，之后再浏览器内检查元素 看样式有没有挂载上

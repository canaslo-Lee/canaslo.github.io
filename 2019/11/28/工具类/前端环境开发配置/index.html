前端环境开发配置

阅读量: 李宜珂 2019-11-28 19:57:50
Categories： Tags： 工具类

先下载通讯工具哟

前端基础软件

建议所有软件从官网下载

浏览器：

  1. 谷歌 chrome 官网：https://google.cn/chrome/
  2. 火狐 firefox 官网：https://firefox.com/
  3. ie win系统自带

node(附带npm包):

​ 选择长期支持版，最新版容易出bug 官网地址：https://nodejs.org/zh-cn/

​ 下载完成后 win+r 输入cmd 回车

​ 运行 命令

npm --version
node --version

如果两个都能显示版本号则安装成功

git：

选择默认版本： 官网地址：https://git-scm.com

进入官网后下载git 下载位置在右侧的电脑中 download开头的，download后边跟的是版本号

下载成功后cmd运行命令

git --version

显示版本号则安装成功

根据公司需要确定是安装 sourcetree 还是git小乌龟（tortoiseGit）

sourcetree官网地址： https://www.sourcetreeapp.com/

tortoiseGit 官网地址： https://tortoisegit.org/download/

小乌龟建议安装中文插件（英文大佬可忽略）：在官网内根据自身电脑选择对应版本的小乌龟和语言包

小乌龟安装的时候先下一步安装小乌龟；安装完毕后，安装中文包；桌面右键选择tortoiseGit的settings进入后右上角的下拉框选择中文重启即可生效

yarn：

有的公司使用yarn 替代npm

yarn官网地址： https://www.yarnpkg.com/zh-Hans/

进入之后选择安装yarn 下载完成后直接安装即可

cnpm：

这个是淘宝镜像；有的安装包在下载的时候会用到；根据需要下载

官网地址：npm.taobao.org

编辑器：

  1. sublime： http://www.sublimetext.com/
  2. vscode： https://code.visualstudio.com/
  3. webstorm： http://www.jetbrains.com/webstorm/
  4. hybuilder： https://www.dcloud.io/hbuilderx.html

sublime 在我的blog中有破解方法

postman：

用于接口测试

官网地址：https://www.getpostman.com/

ps：

photoshop：官网地址： https://www.adobe.com/cn/products/photoshop.html

04.生命周期

阅读量: 静思 2019-11-24 22:07:05
Categories： vue Tags： vue基础

生命周期是什么

  • 实例从创建到销毁的过程
  • 从开始创建、初始化数据、编译模板、挂载 DOM ->渲染、更新->渲染、销毁等一系列过程称为声明周期

生命周期的作用

  • 生命周期中有多个事件钩子，在控制整个实例的过程时更容易形成逻辑闭环

生命周期

  • beforeCreate 创建前
    • 数据监听和初始化事件未开始
  • created 创建后
    • 完成数据监听、属性、方法的运算、初始化事件、$el 属性未显示
  • beforeMount 加载前
    • 挂载开始前调用，相关的 render 函数首次执行，实例已完成以下配置：
      • 编译模板
      • 将 data 中的数据和模板生成 html 且未将 html 挂载到页面
  • Mounted 加载后
    • el 被新创建的 vm.$el 替换并挂载到实例上之后调用；实例以完成以下配置：
      • 编译好的 html 内容替换 el 属性指向的 DOM 对象。完成模板中 html 页面渲染
      • 此过程中进行接口数据交互
  • beforeUpdate 更新前
    • 数据更新前调用，在虚拟 DOM 重新渲染和打补丁之前发生
    • 可在该狗子中进一步更改状态，不触发附加渲染过程
  • updated 更新后
    • 在数据更改导致页面重绘和打补丁之后调用
    • 调用时组件 DOM 已更新，可执行依赖 DOM 的操作
    • 避免在此期间更改状态，可能导致无限更新循环
    • 渲染期间不被调用
  • beforeDestroy 销毁前
    • 实例销毁前调用，实例仍然可用
  • destroyed 销毁后
    • 实例销毁后调用，调用后所有的事件监听器被移除，所有的子实例被销毁
    • 渲染期间不被调用

首次页面加载触发哪些钩子

  • beforeCreate
  • created
  • beforeMount
  • mounted

DOM 渲染在哪个阶段完成

  • mounted
